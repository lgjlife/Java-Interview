## 通配符总结
通配符在很多地方都使用到，但使用都不一样，现总结如下。

### MySQL
|||
|---|---|
|_|下划线，匹配单个字符|
|%|匹配多个字符|

### Linux
|||
|---|---|
|*|匹配多个字符|
|?|匹配单个字符|
### 正则表达式

|||
|---|---|
|*|0或多个匹配|
|+|1个或多个匹配|
|?|0或1个匹配|
